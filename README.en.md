# 基于spring的在线家教管理系统2

#### Description
基于spring的在线家教管理系统2拥有三种角色

- 管理员：会员管理、教师管理、家教列表、发布家教需求、教师接单列表、辅导机构列表、试题列表等
- 教师：登录注册、个人信息修改、查看预约记录
- 学生：发布需求、我的订单、预约教员记录、个人信息修改、登录注册等
